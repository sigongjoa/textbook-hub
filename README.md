# textbook-hub (가칭)

선생님들을 위한 교재 버전관리·공유 웹 SaaS 구상.

## 확정된 방향

- **형태**: 웹 SaaS
- **목적**: 선생님 커뮤니티 형성
- **공유 범위**: 전면 공개 (누구나 fork/사용)
- **Fork 단위**: **블록 단위** (개념·예제·문제·시각자료·풀이 각각)
- **편집 방식**: **한정된 프로세스(constrained operations)** — 자유 편집이 아니라 정해진 변형 오퍼레이션 안에서만 수정
- **AI 역할**: 미정

## 핵심 시나리오

"A 선생님의 개념 설명은 좋으니 가져오고, 문제는 내 학생 수준에 맞게 바꾼다."

→ **Fork(블록)** + **한정된 변형(프로세스)**의 조합으로 달성.
